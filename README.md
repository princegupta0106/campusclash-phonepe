# campusclash-phonepe

🎮 **Campus Clash** — The ultimate campus tournament platform by **NextByte Technologies**.

## Features
- User registration & login (JWT auth)
- Browse upcoming & past tournaments
- Register for tournaments with dummy PhonePe payment gateway
- Admin panel (hardcoded admin email)
- About, Contact, Help pages
- SQLite database, Node.js + Express backend

## Tech Stack
- **Backend:** Node.js, Express, SQLite (sql.js)
- **Frontend:** Vanilla HTML, CSS, JavaScript
- **Auth:** JWT + bcryptjs

## How to Run
```bash
npm install
node server.js
```
Then open **http://localhost:3000**

## Admin Access
Login with the hardcoded admin email to access the admin panel.
